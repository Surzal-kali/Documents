# OSINT Tools Automatable with Python, Bash, or PowerShell

| Tool | Automatable with | Typical use |
|---|---|---|
| Amass | Python/Bash/Pwsh | Subdomain and asset discovery |
| subfinder | Python/Bash/Pwsh | Fast subdomain enumeration |
| theHarvester | Python/Bash/Pwsh | Emails, hosts, domains, people data |
| recon-ng | Python | Modular recon workflows |
| SpiderFoot | Python/Bash/Pwsh | Automated OSINT collection and correlation |
| Shodan | Python/Bash/Pwsh | Internet-exposed asset search |
| Censys | Python/Bash/Pwsh | Host, cert, and search automation |
| urlscan.io | Python/Bash/Pwsh | URL and domain intelligence scans |
| crt.sh / CT logs | Python/Bash/Pwsh | Certificate transparency recon |
| Wayback / CDX | Python/Bash/Pwsh | Historical URLs and content |
| GitHub CLI / GitHub API | Python/Bash/Pwsh | Repo, user, and org OSINT |
| Maltego | Python/Pwsh via transforms | Link analysis and graph enrichment |
| HaveIBeenPwned | Python/Bash/Pwsh | Breach and domain exposure checks |
| intelx | Python/Bash/Pwsh | Search across leaked/public datasets |
| Hunter.io | Python/Bash/Pwsh | Email and domain discovery |

## Practical split

- **Bash** for chaining tools and pipes
- **Python** for API-heavy workflows and data processing
- **PowerShell** for Windows-friendly automation and JSON handling
